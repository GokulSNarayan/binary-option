# Binary Option - Currency Converter & Exchange Rates

### Running Application in Development Mode

> `yarn dev ` or `npm run dev`

### Production build

> `yarn build` or `npm run build`

## Tools used:

- React
- Redux
- Vitejs
- Redux Saga
- Styled Components
- Axios

## API Ref:

- [Exchange Rate API](https://www.exchangerate-api.com/)
